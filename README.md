# 🍳 Chef Notes – Instant Recipe & Smart Cooking App

Chef Notes is a modern web application that helps users cook delicious meals without watching long videos. It provides instant access to chef-curated recipes with clear ingredients, steps, and cooking guidance.

---

## 🚀 Features

- 🔍 Search meals by ingredient or food name  
- 🍽 Chef’s Pick – random recipe on every homepage visit  
- 📂 Browse recipes by category  
- 📖 Detailed recipe view with ingredients and instructions  
- ▶️ YouTube cooking video support  
- 👤 User login & signup using LocalStorage  
- 🎨 Clean, modern and responsive UI  

---

## 🧑‍🍳 How It Works

1. User searches for a dish or ingredient  
2. App fetches data from **TheMealDB API**  
3. Recipes are displayed as interactive cards  
4. Clicking a recipe opens full cooking details  
5. Users can log in to get a personalized experience
